# TypeScript 学习计划

## 学习路线

| 序号 | 主题 | 目标 |
|------|------|------|
| 01 | 基础类型 | 掌握 string, number, boolean, array, tuple, any, unknown, void, never |
| 02 | 接口与类型别名 | 理解 interface 与 type 的区别和使用场景 |
| 03 | 函数 | 函数类型、可选参数、默认参数、重载 |
| 04 | 类 | 类的修饰符、继承、抽象类、实现接口 |
| 05 | 泛型 | 泛型函数、泛型类、泛型约束 |
| 06 | 枚举与字面量类型 | enum、const enum、联合类型、类型字面量 |
| 07 | 类型守卫与缩窄 | typeof、instanceof、in、自定义类型守卫 |
| 08 | 工具类型 | Partial, Required, Pick, Omit, Record 等内置工具类型 |
| 09 | 装饰器 | 类装饰器、方法装饰器、属性装饰器 |
| 10 | 模块与命名空间 | ESM/CJS 模块、命名空间、声明文件 |
| 11 | 异步编程 | Promise 创建/链式/组合器、async/await、错误处理、重试/超时/并发限制、async 迭代器 |

## 运行方式

```bash
# 在项目根目录下运行任意示例
npx ts-node 01-typescript/01-basic-types/index.ts
```
